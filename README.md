# Crud_inazuma
Proyecto de entorno servidor.

Un CRUD de equipos y jugadores de Inazuma Eleven en el que cada usuario puede gestionar su equipo y sus jugadores además de poder ver los equipos de los demás usuarios.

Tutorial de uso: https://youtu.be/BtswL6HNbYo
